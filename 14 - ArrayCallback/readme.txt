Primeiro aprendemos que callbacks são funções quue são usadas como parametro para outra função.
Neste modulo aprenderemos a usar os callbacks dos arrays e trabalhar com uma syntax mais limpa e com funções prontas do javascript.

são os callbacks = forEach,Map,filter,find,reduce,some,every

forEach - Iterar sobre o array, (função de callback, indice  )
Map - criar um novo array com aquilo que eu queria apartir do array anterior
Find - se uma expressão der true ele indica qual é o elemento e para
filter - de acordo com uma expressão eu crio um novo array com aquilo que quero
every - se todos os elementos forem true o resultado sera true similar ao &&
some - se any of elements forem true o resultado sera true similar ao ||
sort - metodo que organiza os elementos do array cresc:a-b e dec: b-a 
reduce -reduz todos os elementos a um só
